Step 1 — Set up on your computer

Get started with Coach

Coach runs on your computer using Claude Code (Pro or Max). It analyzes your food photos, tracks calories, builds meal plans, and coaches you in real time.

1

Create a folder

This is where Coach keeps your data. Open a terminal and run:

mkdir coach && cd coach
mkdir coach; cd coach
2

Install Coach

Still in your terminal, run these two commands:

claude plugin marketplace add nEmily/health-tracker
claude plugin install coach@health-tracker --scope local
3

Launch Coach

Type claude in the same folder. Coach will introduce itself and help you set up your goals, connect your phone, and get everything running.

Optional: Apple Health step tracking (iOS)

Send your daily steps, distance, and active calories to Coach automatically using an iOS Shortcut.

1

Create the Shortcut

Open Shortcuts on your iPhone. Tap + to create a new shortcut and add these actions in order:

Date > Format Date (Custom: yyyy-MM-dd) > save as today
Find Health Samples: Step Count, today, group by Day > save as steps
Find Health Samples: Walking + Running Distance, today, group by Day > save as distance
Find Health Samples: Active Energy, today, group by Day > save as activeCalories
2

Send to Coach

Add a Get Contents of URL action:

Paste your sync key above first
Method: PUT
Headers: Content-Type = application/json
Body (JSON): steps, distance_mi, activeCalories, date = today, source = apple_health
3

Automate it

Go to Automations tab > Time of Day > 9 PM > Run Shortcut. Turn off "Ask Before Running." Steps sync every night automatically.
